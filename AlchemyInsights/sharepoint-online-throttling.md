# <a name="sharepoint-online-throttling"></a>Limitación de SharePoint Online

SharePoint Online utiliza limitación para mantener un rendimiento óptimo y la confiabilidad del servicio SharePoint Online. La limitación restringe el número de acciones de usuario o llamadas simultáneas (mediante script o código) para evitar el uso excesivo de recursos. Si obtiene una limitación, tenga en cuenta que el 99 % de las veces es debido al código personalizado.
  
Para obtener más información sobre la limitación, vea [Avoid getting throttled or blocked in SharePoint Online](https://go.microsoft.com/fwlink/?linkid=2022019)
