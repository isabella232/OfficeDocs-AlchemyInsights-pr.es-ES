# <a name="buy-azure-active-directory-premium-p2"></a>Comprar Azure Active Directory Premium P2

Para probar o comprar Azure Active Directory Premium P2, siga estos pasos:

1. Vaya a **Facturación** Servicios de compra Seguridad e identidad  >    >  [](https://go.microsoft.com/fwlink/?linkid=2131946)y, a continuación, haga clic **en Azure Active Directory Premium P2**.
Si ha seleccionado la prueba gratuita, complete el proceso siguiendo las instrucciones de la página; de lo contrario, continúe con el paso 2.
1. En la página siguiente, escriba el número de licencias de usuario y el período de pago y, a continuación, haga clic **en Des check-out now**.
1. En la **página Desprotección,** revise la información de precios y, a continuación, haga clic **en Siguiente**.
1. En la página siguiente, escriba la información de pago y dirección y, a continuación, haga el pedido.
1. Escriba su número de teléfono y, a continuación, haga clic **en Enviarme un mensaje** de texto o **llamarme**.
1. Escriba el código de verificación que le hemos enviado y, a continuación, complete el proceso siguiendo las instrucciones de la página.
