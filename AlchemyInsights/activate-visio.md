# <a name="activate-visio"></a>Activar Visio

Antes de que un usuario pueda instalar y activar la aplicación de escritorio Visio de escritorio incluida con su plan Visio Online 2, el usuario debe tener asignada una Visio [de escritorio.](https://docs.microsoft.com/microsoft-365/admin/add-users/add-users)

**Nota** Visio Plan 1 en línea no incluye una versión de escritorio de Visio.

Para activar Visio:

1. Instale Visio desde el [portal de Mi](https://portal.office.com/account#installs) cuenta si aún no está instalado. Para obtener ayuda, consulte [Instalar Visio](https://support.office.com/article/f98f21e3-aa02-4827-9167-ddab5b025710?wt.mc_id=OfficeAdm_ClientDIA_Alchemy1800).
2. Cuando finalice la instalación, abra la aplicación de Visio. Es probable que Visio se active automáticamente y puede empezar a usarlo de inmediato. Si no es así, es posible que se le pida que inicie sesión. Asegúrese de iniciar sesión con la misma cuenta profesional o educativa a la que se le ha asignado una licencia de Visio. Siga las indicaciones para continuar con la activación. 

Para problemas de activación, ejecute el [solucionador de](https://aka.ms/SARA-OfficeActivation-Alchemy) problemas de activación en el equipo afectado.
