# <a name="add-external-email-addresses-to-a-distribution-group"></a>Agregar direcciones de correo electrónico externas a un grupo de distribución

En primer lugar, debe crear un contacto en la organización para la dirección de correo electrónico externa. A continuación, puede agregar el contacto al grupo de distribución. Para obtener más información, vea [Agregar usuarios externos a un grupo de distribución.](https://support.office.com/client/caa0f310-0bb7-48e3-8ad2-cb358b53bbba)

Si desea que los usuarios externos puedan enviar mensajes al grupo de distribución, compruebe que el grupo está configurado para aceptar correo de remitentes externos. Para obtener más información, vea [Delivery management](https://technet.microsoft.com/library/bb124513.aspx#deliverymanagement).
