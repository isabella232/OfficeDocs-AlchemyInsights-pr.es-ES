# <a name="change-your-email-address"></a>Cambiar la dirección de correo electrónico 

Para cambiar la dirección de correo electrónico, necesitará privilegios de administrador.
  
Siga las instrucciones de [Cambiar un nombre de usuario y una dirección de correo electrónico en Microsoft 365](https://docs.microsoft.com/microsoft-365/admin/add-users/change-a-user-name-and-email-address).
