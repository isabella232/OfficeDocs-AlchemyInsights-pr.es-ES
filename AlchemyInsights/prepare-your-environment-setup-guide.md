# <a name="prepare-your-environment-for-microsoft-365-and-office-365-services"></a>Preparar el entorno para Microsoft 365 y Office 365 servicios

La [guía Preparar su entorno le](https://go.microsoft.com/fwlink/?linkid=2005213) ayuda a preparar el entorno de su organización para Microsoft 365 y Office 365 servicios. Independientemente de sus objetivos, hay tareas que deberá completar para garantizar una implementación correcta.

Para evitar problemas al preparar el entorno, se le proporcionan instrucciones paso a paso para conectar su dominio, agregar usuarios, asignar licencias, configurar el correo electrónico con Exchange Online e instalar o implementar aplicaciones Office.
