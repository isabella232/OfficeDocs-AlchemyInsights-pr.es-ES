# <a name="your-users-mailbox-is-being-created"></a>Se está creando el buzón del usuario

Puede tardar hasta una hora antes de que un nuevo buzón esté listo para usarse.
  
Si el buzón de un usuario ha estado "preparándose" durante más de una hora, puede intentar eliminar y volver a crear el usuario, o usar la opción "Let us call you" para hablar con un ingeniero de soporte técnico.
