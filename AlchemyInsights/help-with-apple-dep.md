# <a name="help-with-apple-dep"></a>Ayuda con Apple DEP

Si un dispositivo DEP no puede inscribirse y mfa (autenticación multifactor) está habilitado, deshabilite MFA. Mfa no se admite actualmente para la inscripción de DEP.
