# <a name="how-to-move-a-onenote-notebook"></a>Cómo mover un bloc de notas OneNote de datos

* Para mover un bloc de notas OneNote a OneDrive, vaya a Archivos  >  **compartir**  >  **mover blocs** de notas en OneNote 2016. Vea [Mover un bloc OneNote a OneDrive](https://support.office.com/article/Move-a-OneNote-notebook-to-OneDrive-0af0a141-0bdf-49ab-9e50-45dbcca44082) para obtener más información.

* Para obtener información adicional sobre los procedimientos recomendados mientras se prepara para mover blocs de notas, vea [OneNote sincronización de procedimientos recomendados](https://support.microsoft.com/help/2819334/onenote-syncing-best-practices).

* [Mover un OneNote para Windows bloc](https://support.office.com/article/Move-a-OneNote-for-Windows-notebook-that-you-ve-shared-with-others-56c7659e-1850-49a6-8874-e2db6b440cd4) de notas que ha compartido con otros proporciona más información sobre cómo mover blocs de notas compartidos.
