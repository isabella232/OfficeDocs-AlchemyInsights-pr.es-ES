# <a name="unknown-user-appears-in-teams-meetings-or-chats"></a>"Usuario desconocido" aparece en Teams reuniones o chats

Es posible que el usuario se haya eliminado de Microsoft 365. En este caso, intente eliminar el usuario de Teams.  

Intente borrar la siguiente memoria caché Teams se cierra y, a continuación, reinicie Teams:`C:\Users\<USER>\AppData\Roaming\Microsoft\Teams`

Para obtener actualizaciones sobre este problema, vea [Removed user appears as "Unknown user" en Microsoft Teams](https://docs.microsoft.com/MicrosoftTeams/troubleshoot/known-issues/removed-user-appears-as-unknown).
