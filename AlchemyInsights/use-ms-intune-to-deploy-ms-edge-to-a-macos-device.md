# <a name="use-microsoft-intune-to-deploy-microsoft-edge-to-a-macos-device"></a>Usar Microsoft Intune para implementar Microsoft Edge en un dispositivo macOS

Para implementar, configurar, supervisar y proteger una aplicación en un dispositivo macOS, debes agregar la aplicación a Microsoft Intune. La aplicación debe ser de un tipo compatible con Intune y las opciones de administración disponibles en Intune dependerán del tipo de la aplicación. El uso de Intune también significa que no tendrás que usar la herramienta de ajuste de aplicaciones de macOS.

Intune admite Microsoft Edge. Para obtener más información, [vea Agregar Microsoft Edge a dispositivos macOS con Microsoft Intune](https://go.microsoft.com/fwlink/?linkid=2134949).
