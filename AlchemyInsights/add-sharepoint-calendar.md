# <a name="add-a-sharepoint-calendar"></a>Agregar un SharePoint calendario

SharePoint Los calendarios son tipos de lista clásicos que a veces se confunden Microsoft 365 calendarios de grupo.
 
Para crear un calendario SharePoint, siga estos pasos:
 
1.  Selecciona el **Configuración** de engranaje y, a continuación, **selecciona Agregar una aplicación**.
2.  En el **cuadro Buscar una aplicación,** escriba **Calendario** y presione ENTRAR.
3.  Selecciona la aplicación titulada **Calendario** y, a continuación, escribe un nombre para el calendario.

Para obtener más información acerca de Microsoft 365, vea [Learn about Microsoft 365 Groups](https://support.office.com/article/Learn-about-Office-365-groups-b565caa1-5c40-40ef-9915-60fdb2d97fa2).
